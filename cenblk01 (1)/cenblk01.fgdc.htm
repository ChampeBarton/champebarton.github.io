Description:
Abstract:
This dataset contains US Census Bureau 2000 data for the State of Florida. The data contains selected fields from the Summary 1 file that includes information on total population, race, sex, age, and household. This data is at the block level, the smallest unit of census data. Using GIS techniques, it is possible to summarize the data by Block Group, Tract, Traffic Analysis Zone (TAZ), Zip Code Tabulation Area (ZCTA), or County.

Additional demographic data from the Summary 1 is available as supplemental tables that can be joined to Cenblk.shp (see FEATURE ATTRIBUTE TABLES CODES AND VALUES - Supplemental Tables).

Purpose:
This dataset was created to provide a digital represention to be used for analytical and planning purposes.

Process_Step:
Process_Description:
Census Block Boundaries were downloaded by the GeoPlan Center from www.geographynetwork.com/data/tiger2000. The data was downloaded in shapefile format with the following projection parameters: Geographic, Datum: NAD 83, Spheroid: GRS80, Units: Decimal Degrees

The Summary 1 file was downloaded by the GeoPlan Center from Census.gov in delimited text format.

The 2000 SF1 Block Data from Geography Network was downloaded by the GeoPlan Center from www.geographynetwork.com/data/tiger2000 in dbf format.

Geoplan, during the QA/QC process included the following aspects:
(1) joined shapefile with GeoHeader file from Summary 1,
(2) add descript item based on total population,
(3) projected data to FGDL Albers HPGN,
(4) converted AREALAND, AREAWATR, POP100, HU100 fields from character to number,
(5) downloaded 2000 SF1 Block Data from Geography Network,
(6) joined Block Data with shapefile,
(7) dropped duplicate fields (State, County, Tract, Block),
(8) calculated total acres using x-tools for ArcView 3.2,
(9) Added Popdens (Population Density) and PCTMNRTY (Percent Minority Population)
note: for 3 records ACRES = .000 for these records the ACRES were changed to .001.
